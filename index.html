Realtime DDoS Detection System

This project involved developing an AI-powered system for detecting and mitigating Distributed Denial of Service (DDoS) attacks in real-time. The system uses Self-Organizing Maps (SOMs), a type of artificial neural network, to analyze network traffic patterns and identify anomalies that indicate potential attacks.

The implementation leverages Python with Scapy for packet analysis, Mininet for network emulation, and the Ryu controller for SDN (Software-Defined Networking) capabilities. The system was tested against various attack scenarios and demonstrated a 92% detection accuracy with minimal false positives.

ExamWizard2022

ExamWizard2022 is a comprehensive web-based examination administration system designed to streamline the process of creating, scheduling, and managing exams for educational institutions. The platform provides tools for question bank management, automated scheduling, and real-time result processing.

Built with Django and PostgreSQL, the system features role-based access control for administrators, teachers, and students. Key functionalities include automated grading, plagiarism detection, and detailed analytics dashboards. The system reduced exam administration time by 65% for our pilot institution.
